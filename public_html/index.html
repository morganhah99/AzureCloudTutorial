Azure Tutorial

This will be a tutorial for setting up a website using azure.

Git

  1. Git clone a repository that gets the basse setup for docker

    2. $ git clone https://github.com/kaw393939/apacheDocker.git

  2. create seperate directory, I will call it azure.
  3. Copy the apacheDocker contents and paste to the azure directory

    4. Copy these files in apacheDocker

    1. public_html
    2. Dockerfile
    3. dokcer-compose.yml
    4. readme.md
  4. initialize the git repo

    5. $ git init

  5. Go to GitHub and create a new repository
  6. Add github repo to your local git repo

    7. $ git remote set-url --add [repo_name] [repo-url]

  7. add all the files in the directory to be commited

    $ git add .

  8. package the changes in the local repo

    9. $ git commit -m "meaningful message"

  9. Push the git repo changes to remote

    10. $ git push

Azure

  1. Log in to your Azure Microsoft Account

    2. $ docker login azure

  2. Create the docker context for Azure

    3. $ docker context create aci azureCloud

  3. Change the docker context to azureCloud

    4. $ docker context use azureCloud

  4. Build the doccker image from the repo in docker hub

    5. $ docker build -t username/repo_name

  5. Run the docker image from the repo in docker hub

    6. $ docker run -dp 80:80 username/repo

    the port must be 80:80 for it to work

  6. List the running docker containers

    7. $ docker ps

  7. Look at PORTS. Copy the website ip address which can look something like this, XX.XX.XXX.XXX. Paste the website ip address to the browser search bar.

Congratulations, you created a website that the public can access.
