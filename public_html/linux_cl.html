Linux Command Line

  • On Linux or Unix, open a terminal.
  • One Windows, open Git Bash
  • If you do not have Git Bash, install Git
  • print working directory 
    •  $ pwd
  • change directory 
    •  $ cd /home/username
  • ~ = user directory
    • user directory in windows:

      • C:\Users\username

    • user directory in linux:

      • /home/username

    • user directory in mac:

      • /Users/username

  • print the active username 
    •  $ whoami
  • make directory 
    •  $ mkdir "folder"
  • list files and directories on the directory you are on 
    •  $ ls
  • list all files on the directory you are on 
    •  $ ls -a

    This includes hidden files that start with period. Ex: .config

  • list files and directories of a specified directory 
    •  $ ls /home/username
  • Create empty file 
    •  $ touch file.txt
  • open the file using vi text editor 
    •  $ vi file.txt
  • print fle content in the terminal 
    •  $ cat file.txt
  • prints the text in the terminal 
  •  $ echo "this is echo"
  • moves a file to a different directory 
    •  $ mv file.txt ~/file.txt
  • rename a file 
    •  $ mv file.txt file2.txt
  • remove a file 
    •  $ rm file.txt
  • removes empty directory 
    •  $ rmdir folder
  • remove a non-empty directory that contains files and directories 
    •  $ rm -r folder
  • count the number of words in the text file 
    •  $ wc -w text.txt
  • count the number of lines in the text file 
    •  $ wc -l text.txt
