Azure

Create Azure Account

  1. Go to this link to get the student edition of Azure: azure student edition
  2. Click "start free" button
  3. login to your student account or create microsoft student account using your college student email
  4. Go to Microsoft's azure website: Microsoft's azure website
  5. Click the profile icon at the top right corner then click on "Azure Portal"

    6. You should be at the Azure Portal

Docker azure commands

  • To log in to Azure for docker on the terminal

    • $ docker login azure

  • To create the docker context for Azure

    • $ docker context create aci azureCloud
