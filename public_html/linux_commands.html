Linux Commands

Linux is a family of open source Unix like operating systems based on the Linux kernel. The Linux kernel was created by Linus Torvards. This page will help you learn the various Linux commands that are useful for your terminal usage.

  • Open a terminal. If using Windows, open Git Bash.
  • If you do not have Git Bash, install Git
  • Print working directory: 
    •  $ pwd
  • Change directory: 
    •  $ cd /home/username
  • ~ = user directory
    • user directory in windows:

      • C:\Users\username

    • user directory in linux:

      • /home/username

    • user directory in mac:

      • /Users/username

  • Print the active username: 
    •  $ whoami
  • Make directory: 
    •  $ mkdir "folder"
  • List files and directories on the directory you are on: 
    •  $ ls
  • List all the files and directories on the directory you are on: 
    •  $ ls -a

    This includes hidden files and directories that start with period. Ex: .config

  • List files and directories of a specified directory: 
    •  $ ls /home/username
  • Create an empty file: 
    •  $ touch file.txt
  • Open the file using the vi text editor: 
    •  $ vi file.txt
  • Display the file contents: 
    •  $ cat file.txt
  • Display the text: 
  •  $ echo "this is echo"
  • Move a file to a different directory: 
    •  $ mv file.txt ~/file.txt
  • Rename a file: 
    •  $ mv file.txt file2.txt
  • Remove a file: 
    •  $ rm file.txt
  • Remove an empty directory: 
    •  $ rmdir folder
  • Remove a non-empty directory that contains files and directories: 
    •  $ rm -r folder
  • Count the number of words in the text file: 
    •  $ wc -w text.txt
  • Count the number of lines in the text file: 
    •  $ wc -l text.txt
